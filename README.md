## Project Overview

The **CSE0001 – Digital Literacy** course at VIT Bhopal University takes a practical, portfolio-based approach to assessment. Rather than written examinations, students act as **Digital Ambassadors** and demonstrate competency through five hands-on tasks — from designing an infographic to investigating a real-world cybercrime case.

This project covers five interconnected themes:

1. Visual communication and infographic design (Canva)
2. Professional online presence (GitHub, LinkedIn, ResearchGate)
3. Coding practice and collaborative tools (HackerRank, Google Forms/Sheets)
4. Email etiquette and responsible digital communication
5. Cybercrime research and analysis (WannaCry ransomware)

All deliverables are housed in a structured GitHub repository following open-source folder conventions.

## Module Summaries

### Task 1 — Digital Literacy Awareness Infographic
**Tool:** Canva | **Folder:** `task-1-presentation/`

A one-page infographic titled "Digital Literacy Guide" was created using Canva, covering five core themes: digital literacy basics, useful tools, safe online behaviour, professional presence, and email etiquette.

---

### Task 2 — Professional Online Presence
**Platforms:** GitHub · LinkedIn · ResearchGate | **Folder:** `task-2-online-presence/`

Professional profiles were set up on GitHub (technical portfolio), LinkedIn (academic and career credentials), and ResearchGate (academic footprint). The project repository follows standard open-source folder structure.

---

### Task 3 — Coding and Collaboration Platforms
**Platforms:** HackerRank · Google Forms · Google Sheets | **Folder:** `task-3-platforms/`

A HackerRank C++ challenge ("For Loop") was completed with a perfect score (10/10). A five-question Digital Literacy Awareness Quiz was also built in Google Forms and connected to Google Sheets for automatic response collection.

---

### Task 4 — Email Etiquette and Digital Communication
**Deliverables:** 2 Professional Emails · Social Media Checklist | **Folder:** `task-4-email-etiquette/`

Two professional emails were drafted — one requesting an assignment extension and one for an internship enquiry. A Social Media Do's and Don'ts checklist was also prepared covering responsible online behaviour.

---

### Task 5 — Cybercrime Research: WannaCry Ransomware
**Case Study:** WannaCry Ransomware Attack (May 2017) | **Folder:** `task-5-cybercrime/`

A detailed case study on the WannaCry ransomware attack, which compromised 200,000+ systems across 150+ countries and caused over $4 billion in damages. The study covers the attack vector (EternalBlue / MS17-010), real-world impact on NHS hospitals, and recommended prevention measures.

---

## Conclusion

The five tasks offered a practical and integrated learning experience — developing skills in visual design, professional identity, coding, communication, and cybersecurity awareness. These form a foundation for every subsequent module in the Cyber Security programme.
